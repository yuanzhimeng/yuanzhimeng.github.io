安装前准备

  • git -v 检查是否安装git
  • npm -v 检查是否安装node.js

安装hexo

  1. 安装hexo npm install -g hexo-cli
  2. 新建一个文件夹， 进入文件夹
  3. 初始化仓库 git init
  4. 初始化hexo hexo init
  5. 启动服务 hexo server
  6. 打开浏览器 http://localhost:4000/
